# VideoNode

视频工厂 —— AI 视频内容生产基地。

## 视频类型

### 宣传/资讯视频
品牌宣传、产品介绍、行业资讯等对外传播视频。

### 知识点视频
技术教程、概念讲解、操作演示等知识分享视频。

## 工作流程

1. 在 [Issues](https://github.com/sleep2agi/VideoNode/issues) 中创建视频选题
2. 分配负责人，排期制作
3. 通过 PR 提交脚本/素材/成品
4. Review 后发布

## 目录结构

```
promo/        # 宣传/资讯视频
knowledge/    # 知识点视频
templates/    # 视频模板和规范
```
