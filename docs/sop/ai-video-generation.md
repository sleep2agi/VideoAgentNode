# AI 视频生成 SOP

## 工具

- [Dreamina](https://www.capcut.com/ai-tool/dreamina) — 文生图、图生视频

## 生成流程

### 1. 准备 Prompt

- 根据脚本画面描述编写 prompt
- 明确风格、构图、色调
- 示例：`"一位工程师在终端前编写代码，赛博朋克风格，16:9，高清"`

### 2. 生成策略

- **小批量优先**：先生成 2-3 张验证效果，满意后再批量
- **迭代优化**：根据结果调整 prompt 关键词
- **Credits 管理**：生成前检查 `dreamina user_credit`，避免中途中断

### 3. 生成模式

| 模式 | 用途 | 说明 |
|------|------|------|
| text-to-image | 生成静态画面 | 用于封面、分镜插图 |
| image-to-video | 静态画面动态化 | 输入图片 → 输出 3-5s 视频片段 |
| text-to-video | 直接生成视频 | 适合简单动作场景 |

### 4. 输出规范

- 分辨率：1920x1080（横屏）或 1080x1920（竖屏）
- 格式：PNG（图片）、MP4（视频）
- 命名：`scene_01_desc.png`、`scene_02_desc.mp4`

### 5. 常见问题

- **人物一致性差** → 使用参考图 + 固定 seed
- **中文渲染模糊** → 后期用字幕叠加，不在生成时加文字
- **动作不自然** → 拆分为多个短片段，后期拼接
